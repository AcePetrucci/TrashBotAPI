# TrashBotAPI

An API for my other project, TrashBot Discord bot.

## Built With
 - [NestJS](https://github.com/nestjs/nest)
 - [GraphQL](https://graphql.org)
 - [TypeGraphQL](https://typegraphql.ml)
 - [Mongoose](https://github.com/Automattic/mongoose)

## Installation
```bash
$ npm i
```

## Running the Bot
```bash
# start
$ npm run start
```
